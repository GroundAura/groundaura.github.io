    Commands

    Here's a list of all the chat commands you can use on my Twitch stream. Click on any !commands to copy them to your clipboard.

    I use Streamer.bot, StreamElements, Ko-fi Stream Alerts, LurkBait, and a chatbot account to manage my chat commands.

    StreamElements commands can also be found on my StreamElements profile page.

    Quotes

    Name Command(s) Description Restrictions Bot Reply Bot
    Get Quote (By Id) !quote <id>, !quote get <id>, !quoteget <id>, !getquote <id> Attempts to get a quote from the quote list with the given Id. WIP
    5s    		 Quote #<id>: " <quote> " - <user> Streamer.bot (@auragolem)
    Get Quote (Latest) !quote latest, !quote get latest, !quoteget latest, !getquote latest Gets the latest added quote from the quote list. WIP
    5s    		 Quote #<id>: " <quote> " - <user> Streamer.bot (@auragolem)
    Get Quote (By Term) !quote <term>, !quote search <term>, !quotesearch <term>, !searchquote <term> Attempts to get a quote from the quote list that includes the given search term. WIP
    5s    		 Quote #<id>: " <quote> " - <user> Streamer.bot (@auragolem)
    Get Quote (Random) !quote, !quote random, !quoterandom, !randquote Gets a random quote from the quote list. WIP
    5s    		 Quote #<id>: " <quote> " - <user> Streamer.bot (@auragolem)
    Add Quote !quote add <user> <quote>, !quoteadd <user> <quote>, !addquote <user> <quote> Adds a quote to the quote list. WIP
    Mods, VIPs
    5s    		 Added Quote #<id>: " <quote> " - <user> Streamer.bot (@auragolem)
    Edit Quote !quote edit <id> <quote>, !quoteedit <id> <quote>, !editquote <id> <quote> Lets you edit a quote in the quote list. Allows you to fix formatting of the quote text. You can type latest instead of a specific id to get the latest quote. WIP
    Mods
    5s    		 Quote #<id> Updated: " <quote> " - <user> Streamer.bot (@auragolem)
    Hide Quote !quote hide <id>, !quotehide <id>, !hidequote <id> Flags a quote from the quote list to be hidden, so it no longer appears when getting quotes. A soft delete for moderation purposes. You can type latest instead of a specific id to get the latest quote. WIP
    Mods
    5s    		 Quote #<id> hidden. Streamer.bot (@auragolem)
    Delete Quote !quote delete <id>, !quotedelete <id>, !delquote <id> Permanently removes a quote from the quote list. You can type latest instead of a specific id to get the latest quote. WIP
    Admins
    5s    		 Quote #<id> removed. Streamer.bot (@auragolem)
    Quote List !quotes, !quotelist Posts a link to a list of the streamer's quotes. WIP Public quote list is still WIP Streamer.bot (@auragolem)
